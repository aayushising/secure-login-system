# Secure Login System with Password Hashing and SQL Injection Prevention

## Project Description
This project demonstrates a secure authentication system that protects user credentials and prevents SQL injection attacks.

## Features
- User Registration
- Secure Login Authentication
- Password Hashing using SHA-256
- SQL Injection Prevention using Parameterized Queries
- SQLite Database Storage

## Technologies Used
- Python
- SQLite
- Hashlib

## Security Techniques Implemented
1. Password Hashing
Passwords are hashed using SHA-256 before storing in the database.

2. SQL Injection Prevention
Parameterized queries are used instead of raw SQL queries to prevent injection attacks.
